Projects

ArtFolio

ArtFolio is a full-stack portfolio tracker that records historical portfolio snapshots, manages a simple event calendar, tracks recent ETF closes, and offers a beginner-friendly investing guide. Users can upload Excel files of company weights, view value and weight trends over time, edit or delete snapshots, monitor SPY/QQQ closes, plan events on a calendar, and read core investing principles.

Full-Stack Forum Application

A modular web forum built with React, TypeScript, Go, and SQLite. I designed separate frontend and backend layers and a hierarchical relational data model for topics, posts, and comments. The RESTful APIs support full CRUD operations with input validation, structured error handling, and proper HTTP status codes. I configured secure communication with CORS middleware and applied role-based access control and least-privilege principles to harden the application.
